cascading style sheet atau CSS adalah pemograman berbasis aturan yang di kembangkan oleh World Web consortium(W3C) pada tahun 1966. CSS dikembangkan karena HTML yang berfungsi untuk memformat halaman web tidak bisa menambahkan tata letak tingkat lanjut misalnya, pada HTML cersi 3.2, sydah di perkenalkan tag agar garua teks bia di ubah, namun, web designer amatir menyalahgunakannya dan tidak ada opsi full back seperti yang sekarang tersedia di css, Jadi , pilihan terbaiknya adalah tetap memisahkan pemformatan (HTML) dan penataan gaya (CSS)
